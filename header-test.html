Header Visibility Test

This page helps test if the header banner is visible against different backgrounds.

White Background Section

This section has a pure white background to test header contrast.

Light Background Section

This section has a light gray background to test header visibility.

Dark Background Section

This section has a dark background to test header visibility in dark mode.

Instructions

1. Open the React app at http://localhost:8081/

2. Check if the header is clearly visible

3. Scroll to see if the header remains visible

4. Test both light and dark themes

5. Test on mobile by resizing the window
